# freebuf_modify_pic
支持t00ls.com文章图片放大&去除 freebuf.com 的文章部分的图片末尾追加的 `!small`  ，让图片直接显示最佳尺寸而不是缩小版的，不需要点击放大查看，方便查看文章。
